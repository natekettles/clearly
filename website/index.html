Clearly Markdown

A clean, native markdown editor for Mac. Write with syntax highlighting, preview instantly, and get back to what matters.

Download for macOS GitHub

v1.4.0 · Requires macOS Sonoma or later

Features

Native macOS

Built with SwiftUI and AppKit. Fast launch, low memory, feels right at home on your Mac.

Syntax Highlighting

Headings, bold, italic, links, code blocks, and more — highlighted as you type.

Instant Preview

Full GitHub Flavored Markdown support, including Mermaid diagrams.

Side-by-Side

Edit and preview at the same time. Scrolling stays in sync so you always see what you're working on.

PDF Export

Export your document as a PDF or send it straight to a printer. Page breaks just work.

Format Shortcuts

Cmd+B for bold, Cmd+I for italic, Cmd+K for links. The keyboard shortcuts you expect.

QuickLook

Preview markdown files right in Finder. Press Space on any .md file.

Light & Dark

Follows your system appearance or set it manually. Looks great either way.

Open Source

Free and open source on GitHub. See how it works, file issues, contribute.
