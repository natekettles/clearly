Clearly Markdown

A native markdown editor and document workspace for Mac. Open folders, browse your files, write with syntax highlighting, and preview instantly.

Mac App Store Direct Download GitHub

v1.11.0 · Requires macOS Sonoma or later

Features

File Explorer

Open folders and browse your markdown files in a sidebar. Bookmark locations, see recents, create and rename files without leaving the app.

Document Outline

A navigable outline of every heading in your document. Click to jump. Toggle with ⇧⌘O.

Native macOS

Built with SwiftUI and AppKit. Fast launch, low memory, feels right at home on your Mac.

Syntax Highlighting

Headings, bold, italic, links, code blocks, and more — highlighted as you type.

Instant Preview

Full GitHub Flavored Markdown support, including Mermaid diagrams and KaTeX math.

Editor & Preview Toggle

Switch between writing and reading with ⌘1 and ⌘2. Scroll position stays in sync across both.

PDF Export

Export your document as a PDF or send it straight to a printer. Page breaks just work.

Format Shortcuts

Cmd+B for bold, Cmd+I for italic, Cmd+K for links. The keyboard shortcuts you expect.

Scratchpad

A menubar app with a global hotkey for jotting things down. Capture a thought without opening a full document.

QuickLook

Preview markdown files right in Finder. Press Space on any .md file.

Light & Dark

Follows your system appearance or set it manually. Looks great either way.

Open Source

Free and open source on GitHub. See how it works, file issues, contribute.
